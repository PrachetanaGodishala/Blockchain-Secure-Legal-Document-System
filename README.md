# Blockchain Secure Legal Document System

## 📌 Project Overview
This project is a web-based application that uses blockchain technology to securely store and verify legal documents. It ensures document integrity, transparency, and tamper-proof storage.

## 🎯 Objective
To provide a secure and decentralized system for storing legal documents using blockchain and smart contracts.

## 🛠 Technologies Used
- Python (Django)
- Blockchain (Ethereum)
- Solidity (Smart Contracts)
- HTML, CSS (Frontend)
- SQLite (Database)

## ⚙️ Features
- Upload legal documents
- Store document details on blockchain
- Verify document authenticity
- Secure and tamper-proof storage
- User-friendly interface

## 📂 Project Structure
- VaultApp → Django application
- templates → HTML pages
- static → CSS and images
- Smart Contracts → Solidity files (.sol)

## 🚀 How to Run
1. Install Python and Django
2. Run command:
   python manage.py runserver
3. Open browser:
   http://127.0.0.1:8000/

## 📌 Conclusion
This project enhances security and trust in document management using blockchain technology.
